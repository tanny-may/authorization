## Запуск проекта
1. Клонировать репозиторий
   ```sh
   git clone https://github.com/tanny-may/authorization
   ```
2. Установить пакеты
   ```sh
   npm install
   ```
3. Запустить сайт
   ```sh
   npm run dev
   ```
